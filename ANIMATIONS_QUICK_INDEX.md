## 🚀 Getting Started Right Now

### 1. See It In Action (Immediate)
```bash
# Open in browser
premium-construction-example.html
```

### 2. Copy the Files (< 1 minute)
- `animations.css` → Your `/css/` folder
- `animations.js` → Your `/js/` folder

### 3. Link in HTML (2 minutes)
```html
<head>
  <link rel="stylesheet" href="css/animations.css">
</head>
<body>
  <!-- Your content -->
  <script src="js/animations.js"></script>
</body>
```

### 4. Add Classes (5 minutes)
```html
<h2 class="scroll-animate">Title</h2>
<button class="btn btn-primary">Action</button>
<div class="scroll-animate-stagger">...</div>
```

### 5. Done! ✅
Your animations are now live.
